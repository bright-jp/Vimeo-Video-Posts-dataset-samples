この **Vimeo Video Posts Dataset Sample** には **1,000件のレコード** が含まれており、**Bright Data API** を使用して抽出されています。

## Data Points

| Field Name | Description |
|------------|-------------|
| `video_id` | 各動画投稿の一意の識別子です。 |
| `title` | 動画投稿のタイトルです。 |
| `url` | Vimeo上の動画投稿への直接URLです。 |
| `uploader` | 動画をアップロードしたユーザーまたはエンティティの名前です。 |
| `video_url` | 埋め込みまたは視聴のための動画プレーヤーへのURLです。 |
| `video_length` | 動画の長さ（秒）です。 |
| `views` | 動画が視聴された回数です。 |
| `likes` | 動画が獲得した「いいね」の数です。 |
| `collections` | 動画が含まれるコレクション数です。 |
| `comments` | 動画に投稿されたコメント数です。 |
| `data_posted` | 動画が投稿された日時です。 |
| `description` | 動画コンテンツの簡単な説明または要約です。 |
| `tags` | 動画に関連付けられたキーワードまたはタグです。 |
| `uploader_url` | アップローダーのプロフィールまたはページへのURLです。 |
| `preview_image` | 動画のプレビュー画像またはサムネイルへのURLです。 |
| `avatar_img_uploader` | アップローダーのアバターまたはプロフィール画像へのURLです。 |
| `related_videos` | Vimeoによって提案される関連動画のリストです。 |
| `license` | 動画が公開されているライセンスの種類です。 |
| `license_info` | 動画のライセンスに関する詳細情報です。 |
| `uploader_id` | アップローダーの一意の識別子です。 |
| `video_quality` | 動画の品質または解像度です。 |
| `height` | 動画の高さ（ピクセル）です。 |
| `width` | 動画の幅（ピクセル）です。 |
| `music_track` | 動画で使用されている音楽トラックに関する情報です。 |
| `transcript` | 利用可能な場合、動画の音声内容のテキスト文字起こしです。 |


## Use Cases

### 1. Content Trend Analysis
Vimeo上の動画コンテンツのトレンドを分析し、人気のトピックや新たなトレンドを特定します。これはコンテンツ制作戦略の策定に役立ちます。

### 2. Social Media Research
動画のエンゲージメント指標に関する調査を実施し、オーディエンスの嗜好を理解して動画マーケティング施策の改善に活用します。

### 3. Competitive Analysis
競合の動画パフォーマンス指標を分析し、自社の動画マーケティング戦略を洗練させ、視聴者エンゲージメントの向上を図ります。
