## Data Points

| Field Name | Description |
|------------|-------------|
| `video_id` | Unique identifier for each video post. |
| `title` | The title of the video post. |
| `url` | Direct URL to the video post on Vimeo. |
| `uploader` | Name of the user or entity who uploaded the video. |
| `video_url` | URL to the video player for embedding or viewing. |
| `video_length` | Duration of the video in seconds. |
| `views` | Number of times the video has been viewed. |
| `likes` | Number of likes the video has received. |
| `collections` | Number of collections the video is part of. |
| `comments` | Number of comments posted on the video. |
| `data_posted` | The date and time when the video was posted. |
| `description` | A brief description or summary of the video's content. |
| `tags` | Keywords or tags associated with the video. |
| `uploader_url` | URL to the uploader's profile or page. |
| `preview_image` | URL to the preview image or thumbnail of the video. |
| `avatar_img_uploader` | URL to the uploader's avatar or profile image. |
| `related_videos` | List of related videos suggested by Vimeo. |
| `license` | Type of license under which the video is published. |
| `license_info` | Detailed information about the video's license. |
| `uploader_id` | Unique identifier for the uploader. |
| `video_quality` | Quality or resolution of the video. |
| `height` | Height of the video in pixels. |
| `width` | Width of the video in pixels. |
| `music_track` | Information about any music track used in the video. |
| `transcript` | Text transcript of the video's audio content, if available. |
